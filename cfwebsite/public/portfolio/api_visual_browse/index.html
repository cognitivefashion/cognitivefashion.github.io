Cognitive Fashion

The IBM Research cognitive fashion project aims to build a suite of APIs for the fashion industry primarily leveraging natural language and image understanding.

The usecases are targeted towards end consumers, online retailers, buyers, merchandisers and designers.

Cognitive Fashion

Visual Browse

Show me more like this

For any given image in the product catalog find other similar images in the product catalog purely based on visual appearance.

A user clicks on an existing apparel/accessory image in the catalog and Visual Browse will fetch and display similar looking products from the catalog. Visual Browse does not need any meta information and search is based only on images. The deep learning based CNN model has been fine tuned to the domain of fashion.

API documentation | Demo


Use cases

  1. Automatically enable other similar looks feature on the website.
  2. Useful to recommend alternate items when an item goes out of stock.
  3. This is a more interactive way of browsing the catalog based on visual search. It essentially recreates the experience of asking Show me more dresses like this.
  4. Encourages serendipitous buying and sticky behavior on the website.
